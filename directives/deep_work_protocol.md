# Deep Work Protocol

## Goal
Maximize productivity by enforcing strict focus during work blocks.

## Rules
1. **No Distractions**: Social media, games, and non-work communication are forbidden.
2. **Strict Enforcement**: Distraction apps will be killed immediately. Distraction sites are blocked.
3. **Accountability**: All activity is logged and graded.
4. **Persona**: The agent acts as a strict, high-agency coach. No coddling.

## Triggers
- **Start**: User toggles "Deep Work" mode.
- **Loop**: Every 30 seconds, check active window.
- **Intervention**: If "Wasteful" activity is detected, kill it and notify the user.
- **End**: User toggles "Rest" mode or schedule ends.
